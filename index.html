Introduction

Rainy bot is an utility bot. You can find informations about your favorite Anime , predict your nationality based on your name , and more features!

    • Commands

    >
    Command Function Usage
    *help Showing help *help
    *cat Showing cat image *cat
    *dog Showing dog image *dog
    *invert Invert discord user's avatar *invert @discord user
    *osu Show osu player information *osu [player name]
    *osurecent Osu recent play from user *osurecent [player name]
    *youtube Fake YouTube comment image *youtube @discord user [text]
    *avatar Show user's avatar *avatar @discord user
    *serverinfo Show server info *serverinfo
    *channelinfo Show channel info *channelinfo
    *kick Kick member *kick @discord user
    *addchannel Add channel *addchannel [Channel name]
    *removechannel Remove channel *removechannel #discord text channel
    *ban Ban user *ban @discord user [reason]
    *purge Clear messages *purge [total messages]
    *translate Translator *translate [language source]-[language destination] [text]
    *weather Show weather *weather [city]
    *voting Voting *voting [text]
    *github Github user info *github [username]
    *wikipedia Search wikipedia *wikipedia [query]
    *encodebinary Encode binary *encodebinary [text]
    *decodebinary Decode binary *decodebinary [binary]
    *encodebase64 Encode base64 *encodebase64 [text]
    *decodebase64 Decode base64 *decodebase64 [text]
    *anime Show anime information *anime [anime title]
    *bored Find something to do if you are bored *bored
    *joke Joke *joke
    *koala Show koala image *koala
    *panda Show panda image *panda
    *roleinfo Show information about role *roleinfo [role name]
    *predictnamecountry Predict your nationality based on your name *predictnamecountry [your name]
    *predictnamegender Predict your gender based on your name *predictnamegender [your name]
